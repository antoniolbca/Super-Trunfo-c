#  Super Trunfo em C

Simulação do cadastro de cartas do jogo Super Trunfo, utilizando dados de cidades e cálculos estatísticos.

##  Conceitos aplicados

- Entrada e saída de dados (`scanf` e `printf`)
- Variáveis compostas
- Cálculos matemáticos
- Conversão de valores (PIB em bilhões para reais)
- Organização de dados

##  Funcionalidades

- Cadastro de duas cartas
- Cálculo de densidade populacional
- Cálculo de PIB per capita
- Exibição formatada dos dados

##  Tecnologias

- Linguagem C
